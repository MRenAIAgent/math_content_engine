1. **Simplify each side** (calculate each player's stats)

## 2.4 Use a General Strategy to Solve Linear Equations

### The 5-Step Championship Strategy ğŸ†

Think of solving equations like a basketball game plan:

1. **SIMPLIFY** (Pre-game warmup) - Distribute and combine like terms
2. **COLLECT VARIABLES** (Offense) - Get all variable terms on one side
3. **COLLECT CONSTANTS** (Defense) - Get all constants on the other side
4. **ISOLATE** (The Shot) - Divide or multiply to get the variable alone
5. **CHECK** (Review the Game Film) - Substitute solution into original equation

### Types of Solutions - Game Outcomes

| Solution Type | What It Means | Basketball Analogy |
