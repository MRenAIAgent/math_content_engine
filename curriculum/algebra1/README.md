# Algebra 1 Curriculum

This folder contains curriculum standards and topic guides for Algebra 1 as taught in US public schools, aligned with Common Core State Standards.

## Contents

- `common_core_standards.md` - Complete list of Common Core Algebra standards
- `curriculum_map.md` - Unit-by-unit topic breakdown with lesson sequences
- `topics_by_difficulty.md` - Topics organized by difficulty level
- `animation_topics.md` - Topics suitable for math animation generation
