## Common Struggle Points

Topics where students frequently need extra support:

### High Difficulty
1. **Factoring ax² + bx + c** (when a ≠ 1)
2. **Word problems with systems**
3. **Completing the square**
4. **Rational expressions operations**
5. **Radical equation solving**

### Medium-High Difficulty
1. **Equations with variables on both sides**
2. **Systems by elimination with multiplication**
3. **Graphing quadratics from standard form**
4. **Exponential word problems**
5. **Finding domain and range**

### Conceptual Hurdles
1. **Understanding function notation f(x)**
2. **Why flip inequality when multiplying by negative**
3. **Why extraneous solutions occur**
4. **Difference between expression and equation**
5. **When to use which form of linear equation**

## Remediation Pathways

### If struggling with solving equations:
1. Review inverse operations
2. Practice two-step equations extensively
3. Use balance model visualization
4. Build to multi-step gradually

### If struggling with graphing:
1. Review coordinate plane basics
2. Practice plotting points
3. Make tables of values
4. Connect slope to real-world rates

### If struggling with factoring:
1. Master multiplication of polynomials first
2. Start with GCF factoring
3. Use area model for trinomials
4. Practice factor by grouping before ax² + bx + c

### If struggling with quadratics:
1. Ensure factoring skills are solid
2. Connect to graphing (roots = x-intercepts)
3. Use technology to explore parabolas
4. Derive quadratic formula step-by-step
